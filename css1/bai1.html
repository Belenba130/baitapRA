What is CSS ?

Cascading Style Sheet (CSS) is used to format the layout of a webpage

With CSS, you can control the color, font, the size of text, the spacing between elements are positioned and laid out, what background image or background colors are to be used, different displays for different devices and screen sizes, and much more!

Tip:The word Cascadingmeans that a style applied to a parent elements will also apply to the children elements within the parent. So, if you set the color of the body text to "blue", all headding, paragraphs, and other text elements within the body will also get them same color (unless you specifi somethings else)!
